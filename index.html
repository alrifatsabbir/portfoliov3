HTML 5

HTML5 is the latest version of the Hypertext Markup Language used
for structuring and presenting content on the web, introducing new
features and improved multimedia support.

CSS 3

CSS3 is the latest iteration of Cascading Style Sheets, bringing new
capabilities for styling and layout on web pages, including enhanced
transitions, animations, and media queries.

JavaScript

JavaScript is a versatile programming language primarily used for creating
interactive and dynamic functionality on web pages, allowing for client-side
scripting and manipulation of HTML elements and CSS properties.

JS ES6

ES6 JS: The latest version of JavaScript that introduced new features
and syntax enhancements, enabling developers to write more concise and
expressive code and improve productivity.

Bootstrap

Bootstrap v5.3: A popular front-end framework that provides a collection
of pre-built responsive components and utilities for quickly and easily
creating modern and mobile-friendly web interfaces.

TailwindCss

Tailwind CSS: A highly customizable CSS framework that focuses
on utility classes, allowing developers to rapidly build user
interfaces by composing classes together.

GitHub

GitHub: A web-based platform for version control and collaboration that
allows developers to host, manage, and share their code repositories,
facilitating teamwork and open-source contributions.

Node JS

Node Js: A JavaScript runtime environment that allows developers
to run JavaScript on the server-side, enabling the development of
scalable and high-performance web applications.

Express JS

Express Js: A minimalistic and flexible web application framework
for Node.js, designed to build web applications and APIs quickly
and easily.

MongoDB

MongoDB: A NoSQL database that provides a flexible,
scalable, and document-oriented approach to storing and
retrieving data, commonly used in modern web applications.

React JS

React Js: A JavaScript library for building user interfaces that allows
developers to create reusable UI components and efficiently
manage the UI state.
